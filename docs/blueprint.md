# **App Name**: Kenya Tube Insights

## Core Features:

- YouTube Statistics Aggregation: Aggregate Kenyan YouTube statistics using the YouTube Data API, collecting data on views, likes, comments, and subscriber counts from Kenyan channels.
- Trending Content Identification: Identify top trending videos and songs specifically within Kenya, based on real-time engagement metrics.
- Interactive Query Tool: Allow users to ask interactive questions about Kenyan YouTube trends and statistics using natural language processing powered by Gemini API. Gemini API serves as a tool that determines how statistics influence answers.
- Statistical Analysis and Reporting: Analyze aggregated data to generate reports on various trends such as most popular music genres, content categories, and emerging creators.
- Data Visualization: Present key YouTube metrics through charts and graphs, enabling users to quickly grasp trends and insights.
- Search and Filtering: Enable users to search and filter Kenyan YouTube channels and videos based on categories, keywords, and date ranges to narrow down statistics.

## Style Guidelines:

- Primary color: Vibrant orange (#FF7043) to reflect Kenya's energetic cultural scene.
- Background color: Light peach (#FFE7D9), a desaturated variant of orange to offer visual softness.
- Accent color: Golden yellow (#FFC107) to highlight important data points and interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif font combining a modern look with warmth, for comfortable reading.
- Use modern and minimalist icons related to video playback and Kenyan culture.
- Design a clean, data-centric layout that is both informative and easy to navigate. Key metrics should be immediately visible, supported by more in-depth analysis available on demand.
- Incorporate smooth transitions and subtle animations to enhance data exploration without overwhelming the user.
